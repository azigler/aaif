## Post formats that work (year-round amplification)

Every time you publish a contribution, post about it and tag **@AgenticAIFdn**:

- **"I just published…"** — link the tutorial / talk recap / video + a one-line
  takeaway.
- **"I'm speaking at [event]…"** — announce upcoming talks; include the badge.
- **"This month I contributed to [project]…"** — brief reflection on what you
  built or taught.
- **"Ask me anything about [project]…"** — invite questions from your audience.
- Use **#AAIFAmbassador** on every post so the cohort can find & boost each other.

> Note: a "useful social thread" is itself a **qualifying contribution** (5 pts)
> when it provides developer value tied to an AAIF project — not just promotion.
> So the amplification layer doubles as the pulse-cadence submission layer (see
> `strategy-top-ambassador.md`).
